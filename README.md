#  Responsive Black & Golden Image Gallery

This is a **fully responsive image gallery** with a **black & golden theme**, implemented using **HTML, CSS, and JavaScript**. It features **category filters**, **lightbox with navigation arrows**, and a smooth hover effect.

---

##  Features

- **Responsive Design**: Works on desktop, tablet, and mobile.
- **Category Filters**: Filter images by `Nature`, `City`, `Portrait`, `Abstract`, or show `All`.
- **Lightbox Popup**: Click an image to view it in a popup with **next** and **previous** navigation.
- **Keyboard Navigation**: Use `ArrowLeft` and `ArrowRight` to navigate, and `Escape` to close the lightbox.
- **Hover Effects**: Image zoom and shadow on hover.
- **Lazy Loading**: Images load efficiently using `loading="lazy"`.
- **Black & Golden Theme**: Stylish dark background with golden highlights for a premium look.

---

##  How to Use

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Click on **filter buttons** to filter images.
4. Click an image to open **lightbox**.
5. Use **‹** and **›** buttons to navigate images.
6. Press **Escape** to close the lightbox.

## Customization

- Add more images to the `images` array in the script.
- Add new categories by updating `data-filter` buttons and `category` field in the images array.
- Change theme colors in CSS by modifying `body` background, `.filter-btn`, `.lightbox-nav`, etc.
